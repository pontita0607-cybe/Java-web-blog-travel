Travel Blog Platform (Java Spring Boot)
A robust blog engine for Thailand travel recommendations, focusing on content management and user interaction.

Core Backend: Developed using Java Spring Boot and Spring Security for authentication (Login/Register/Forgot Password).

Database: Structured with MySQL to handle diverse data types including categories, comments, and reports.

Key Features: >     * Full CRUD operations for blog posts and news updates.

Social engagement features: Like, Comment, and Reporting system.

Dynamic content classification by Categories.
